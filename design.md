# Design System — MetroPT-3 Dashboard

## Vibe
Industrial, data-dense, high-tech. Dark theme. Feels like a real SCADA/monitoring system.
Not a marketing site — a tool that engineers trust.

## Colors
- Background: `#0a0e1a` (deep navy-black)
- Surface: `#111827` (card bg)
- Border: `#1f2937`
- Primary accent: `#3b82f6` (electric blue)
- Danger: `#ef4444` (red alert)
- Warning: `#f59e0b` (amber)
- Safe: `#10b981` (emerald green)
- Text primary: `#f9fafb`
- Text muted: `#6b7280`

## Typography
- Display: `Rajdhani` (industrial, angular)
- Body: `JetBrains Mono` (monospace — data values)
- UI labels: `Inter` fallback

## Layout
- Full-height sidebar navigation
- Main content area with grid panels
- Dense but breathing — 16px base padding, 24px between panels

## Components
- Cards: dark surface, 1px border, subtle glow on hover
- Gauges: SVG arc-based, color shifts green→amber→red
- Status badges: pill-shaped with colored dot
- Charts: dark bg, colored lines, no fill
